# Ebox Build Manual

### 1. Required Components
All the components needed to build a an ebox are listed below:

| Component name                 | Count                |
|--------------------------------|----------------------|
| Raspberry Pi | 1                    |
| Air Sensor(BME280, MS8607, etc)     | 2                    |
| CO2 Sensor(SCD30)       | 2                    |
| Dust Sensor(NextPM, SPS30,etc)|2|
| 4G LTE USB dongle |1|
| Fan|1|
| Raspberry Pi Adapter|1|
